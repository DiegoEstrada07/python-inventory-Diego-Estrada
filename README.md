# python-inventory-Diego-Estrada

A simple Python-based inventory management system that allows users to add, view, update, and remove products. The program uses object-oriented programming, file handling, and basic data structures to simulate a real-world inventory system.

---

## How to Run the Program

### Requirements
- Python 3 installed

### Steps to run 

1. Save the program as `inventory.py`
2. Open a terminal or command prompt
3. Navigate to the folder where the file is located
4. Run the program using:

```bash
python inventory.py
